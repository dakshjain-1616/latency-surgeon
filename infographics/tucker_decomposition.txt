  TUCKER FACTORIZATION: W ≈ A · B
  ┌────────────────────────────────────────────────────────────┐
  │                                                            │
  │   W ∈ R^{768×768}                                        │
  │   SVD: W = U · Σ · Vᵀ                                    │
  │                                                            │
  │        U            Σ_r          Vᵀ                      │
  │   [768 × 768]  ·  [768 × 768]  ·  [768 × 768]           │
  │                        ↓                                  │
  │   Keep only top-r singular values/vectors:                │
  │                                                            │
  │        U_r                        V_rᵀ                   │
  │   [768 × r]   ·  diag(σ₁..σ_r)  ·  [r × 768]           │
  │                        ↓                                  │
  │   Absorb Σ_r into A:                                      │
  │                                                            │
  │     A = U_r · sqrt(Σ_r)    B = sqrt(Σ_r) · V_rᵀ        │
  │     [768 × r]                   [r × 768]                │
  │                                                            │
  └────────────────────────────────────────────────────────────┘
